# Code of Conduct

We want this project to be welcoming and useful. Be respectful.

## The rules

- Treat others with respect. No harassment or hate.
- Assume good intent, especially in text.
- Critique ideas and code, not people.
- No spam, doxxing, or sharing private information.

## Enforcement

If something crosses the line, maintainers may warn, remove content, or ban accounts to protect the community.

## Reporting

If you need to report conduct issues, open a GitHub Discussion titled **“Moderation”** and tag a maintainer.
